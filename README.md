Overview
This project is an interactive Business Intelligence dashboard built using Power BI to analyze retail sales performance, profitability, customer segments, and regional trends using the Superstore dataset.
The dashboard helps identify key business insights such as top-performing categories, profit-driving regions, discount impact on profitability, and sales trends over time.

Project Objectives
Analyze overall sales and profit performance
Identify high-performing product categories and sub-categories
Understand regional sales distribution
Evaluate the impact of discounts on profitability
Build an interactive and user-friendly dashboard for business decision-making

Key Business Insights
Technology category generated the highest profit.
High discounts negatively impacted profitability.
Certain sub-categories showed strong sales but low profit margins.
Regional performance varied significantly across states and customer segments.
